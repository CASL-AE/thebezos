Anytime Liquidity

1% of each transaction gets locked into a liquidity pool. This helps keep funds available for buying and selling.

Holder Residual

1% of each transaction gets put into a pool for disbursement to token holders. The more you hold, the more you get back.

Corporation Kickback

1% of each transaction goes to the beneficiary (Amazon) of the contract. This wallet is currently in a safe deposit box waiting for Amazon.

What is the Bezos?

The Bezos is a token built on the XRP ledger. We are currently planning to add the Bittrue exchange. Our goal is to get Amazon to integrate with the XRP Ledger to accept payments using XRP while rewarding BEZOS token holders.
